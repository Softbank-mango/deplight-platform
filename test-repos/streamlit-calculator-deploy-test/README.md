# Streamlit Calculator Deploy Test

Scientific calculator application built with Streamlit for testing Deplight Platform deployment.

## Features

- **Basic Arithmetic Operations**
  - Addition, Subtraction, Multiplication, Division
  - Power calculations
  - Modulo operations

- **Scientific Functions**
  - Square Root, Square, Cube
  - Trigonometric functions (sin, cos, tan)
  - Logarithmic functions (ln, log₁₀)

- **Quick Calculations**
  - Percentage calculator
  - Circle area calculator
  - Factorial calculator

- **Modern UI**
  - Dark gradient theme
  - Glassmorphism design
  - Responsive layout

## Local Development

```bash
# Install dependencies
pip install -r requirements.txt

# Run the application
streamlit run app.py
```

The application will be available at `http://localhost:8501`

## Deployment

This repository is designed to be deployed automatically through the Deplight Platform. Simply push this repository to trigger the deployment workflow.

## Tech Stack

- Streamlit 1.28.1
- Python 3.11+
- Math library for scientific calculations

## Usage

1. Enter numbers in the input fields
2. Select the desired operation
3. Click the calculate/compute button
4. View the result in the result box
